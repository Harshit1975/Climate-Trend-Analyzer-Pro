# Climate Trend Analyzer Documentation

## Features added
- Custom CSV upload support for real climate datasets
- Multi-tab dashboard for Overview, Trends, Anomalies, Forecast, Data, and About
- KPI metric cards for temperature, rainfall, humidity, CO₂, sea-level, wind speed, and AQI
- Seasonal pattern analysis and correlation heatmaps
- Real-time anomaly detection with adjustable z-score sensitivity
- Forecast comparisons between Linear trend and SARIMAX models
- Model accuracy metrics (MAE and RMSE)
- Download buttons for cleaned data, summary statistics, and forecast output
- Narrative descriptions for analysis insights
- Region selection support for datasets with `location` or `region` columns
