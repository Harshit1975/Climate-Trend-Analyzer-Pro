# Deployment Guide

## Run locally
1. Activate your Python virtual environment.
2. Run `streamlit run app/streamlit_app.py` from the project root.
3. Visit `http://localhost:8501` in your browser.

## Deploy to Streamlit Cloud
1. Push your project to GitHub.
2. Go to https://share.streamlit.io.
3. Create a new app and choose `app/streamlit_app.py` as the main file.
4. Use `requirements.txt` for dependencies.
